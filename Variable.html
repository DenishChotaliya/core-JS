What is Variable?

To store your data in consol is called variable.

How many types of Variable?

There are 3 types of variables

Describe it

VAR - You can do redeclaration & reassistment

var apple = 5 var apple = 2 console.log(apple); //[you will get last value of assistment in var]

LET - You can do only assistment

let banana = 5 banana = 2 console.log(banana); //[you will get last value of assistment in let]

Const - You can not do reassistment & redeclaration bacause it's Block Variable

const kiwi = 5 kiwi = 9 console.log(kiwi); //[ you will get only one value in "block scope"]
